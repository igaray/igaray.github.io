    Ansible

    Getting started with ansible

    controller machine -> managed machine

inventory
    -> machines to be managed by ansible
    -> get through:
        -> inventory file -> /etc/ansible/hosts
            [webservers]
            site01
            site02
            ...
            [production]
            site01
            site02
            db01
            bastion

        -> ansible plug-in
    1. populate /etc/ansible/hosts
    $ ansible site01 --u root -k -m ping
    -u root : username to configure globally, set in /etc/ansible/ansible.cfg
                [defaults]
                remote_user = ...
                remote_port = ...
              this file is overrideable in inventory file per group or server
              to set username in inventory filappend, ansible_ssh_user:
                [webservers]
                site01 ansible_ssh_user=root
    -k      : prompt for password, leave out to use keys
    -m ping : module to run
